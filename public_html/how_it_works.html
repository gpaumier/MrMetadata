How it works

Script

This data is generated by a Pywikibot script, whose code is on GitHub. If the code makes you cry, it's probably normal; five days ago, I had never written any Python code.

HTML pages are generated with Jinja2, using Bootstrap and bits of Font Awesome.

Commons

Commons has a main {{information}} template that already implements machine-readable metadata. When a field is missing, other templates like {{author missing}} and {{source missing}} allow us to track missing metadata.

To fix files missing machine-readable metadata on Commons, it's therefore mostly a matter of:

  • adding the information template to files that don't have it
  • adding missing information to files missing an author or a source

Local uploads

To check local uploads, it's a bit more complex, because of the variety of information templates across wikis. The script needs to go through all file description pages of a wiki, and check for machine-readable metadata by querying the CommonsMetadata extension.

The list of wikis with local uploads was adapted from Robin's uploadconfig tool.
